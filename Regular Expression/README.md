# Regular Expression

## 控制數量

- `*`: 零個或多個
- `+`: 一個以上
- `?`: 零個或一個
- `{n}`: n 個
- `{n,m}`: n 到 m 個

## 跳脫字元

- `\d`: 所有數字
- `\D`: 非數字字元
- `.`: 任何字元
- `\w`: **todo**
- `\W`: **todo**
- `\s`: 空格
- `\S`: 非空格字元

## 特定字元

- `[]`: 括號裡面其中一個
- `[^]`: 不包括
- `[A-Z]`: 大寫字母
- `[a-z]`: 小寫字母
- `[0-9]`: 數字

## 開頭結尾

- `^`: 開頭
- `$`: 結尾

## group

- `()`: 分組
- `(|)`: 包含 string
